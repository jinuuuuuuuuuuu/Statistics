# 1️⃣ 개념 정리 
## 01. 주성분분석

```
✅ 학습 목표 :
* 변수의 차원과 변수의 수를 줄이는 이유를 설명할 수 있다.
* 주성분분석을 이해한다.
```
<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
'차원이 높다'고 하면 정보량이 많으니 얼핏 좋은 것처럼 생각되지만, 실 은 '쓸데없이 많기만 한 상황이 종종 발생
차원 축소: 변수의 수를 줄이는 것

변수의 수를 줄이는 이유
1)	고차원 데이터 해석의 어려움
2)	다중회귀분석에서는 설명변수끼리 강한 상관이 있는 상황을 다중공선성이 있다고 하며, 회귀계수 추정이 불안정해지는 문제가 발생
3)	표본크기 이 충분하지 않은 상황이라면, 회귀계수를 올바르게 추정할 수 없는 문제

주성분 분석
-	PC1: 새로운 축으로 데이터 퍼짐이 가장 커지는 방향으로 설정
-	PC2: PC1과 수직 방향이고 데이터 퍼짐이 가장 커지는 방향의 축
-	원래 각 축에 있 던 정보를 PC1으로 압축해 넣는다는 느낌
-	기여율: 각 주성분이 가진 정보(분산)의 비율
-	누적기여율: '제1부터 제주성분까지 전체 정보의 몇 %가 포함되는지'

주성분 결과
-	PC2 이후는 그때까지 얻은 주성분과 직교하도록 설정
-	새로운 축의 변수와는 무상관이 된다
-	주성분들이 구해지면, 각 주성분의 기여율과 누적기여울을 확인
-	각 주성분이 어떤 축인지를 조사하기 위해 각 주성분의 값과 원래 각 변수의 상관관계를 계산
	주성분부하량 또는 인자부하량
-	주성분 점수: 원래 데이터를 새로운 변수를 이용하여 표로 나타낸 것

인자 분석
-	원래 각 변수에는 소수의 공통인자가 있으며, 이것이 원인이 되어 데 이터의 각 변수가 구성된다는 아이디어에 기반을 둔 방법
-	분석 자체는 관측한 변수 사이의 상관관계에 기반을 두므로, 데이터에서 인과관계를 발견할 수는 없다는 점에는 주의

## 03. 비지도 학습
```
✅ 학습 목표 :
* 비지도 학습을 이해한다.
```
<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
비지도 학습: 정답 데이터가 없으며, 데이터의 배후에 있는 구조를 올바르 게 추출하려는 목적으로 사용

군집분석: 각 데이터가 어떤 군집에 속하는지를 구하는 방법

K 평균
1)	군집 개수를 정하며 시작
2)	각 데이터에 무작위로 군집을 할당하고, 각 군집의 중심 위치(데이터의 평균)를 구하기
3)	데이터마다 가장 근접한 군집을 할당하고 중심 위치를 다 시 계산

계층적 군집화
-	모든 데이터가 다른 군집에 속하는 상태에서 시작하여, 가장 거리가 가까운(비슷한 군집끼리 순서대로 합쳐 나간다
-	덴드로그램 형태
-	군집 개수는 미리 정해지지 않으며, 어느 단계에서 바라보느냐에 따라 달라진다


## 04. 지도 학습 

```
✅ 학습 목표 :
* 지도 학습을 이해한다.
```

<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
지도 학습
-	특정 값일 때, 가 어떤 값이 되는지에 관한 데이터를 바탕으로 그 관계 를 학습해 나가는 방법
-	모형의 출력 과 실제 의 차이를 손실함수로 정식화하고, 이를 최소화하는 파라미터를 구하는 것
-	반응변수가 양적 변수일 때는 회귀, 질적 변수(범주형 변수)일 때는 분류

예측
-	동일 조건에서 얻을 수 있는 미지의 데이터(검증 데이터, 테스 트 데이터)에 대해 설명변수 x로 반응변수를 예측하는 것
-	교차검증: 한쪽 데이터를 사용 해 학습을 진행하고, 나머지 한쪽 데이터는 예측이 얼마나 좋은지 평가하는 데 이용
-	정착화: 모형의 복잡도를 조정하는 요소를 손실함수에 도입할 때

이진 클래스 분류
-	반응변수가 2개의 클래스 A, B일 때
-	1) 위양성: 실제는 음성이나 양성으로 판정 
-	2) 위음성: 실제는 양성이나 음성으로 판정 
-	3) 진양성: 양성을 양성으로 올바르게 판정 
-	4) 진음성: 음성을 음성으로 올바르게 판정
	일반적으로 민감도가 높을수록 특이도는 낮다는 상충 관계

ROC 곡선
-	분류의 예측 성능을 평가
-	이 역치가 작은 값에서 큰 값으로 점점 변할 때의 위양성률(= 1-특이도)을 가로축으로, 민감도를 세로 축으로 하여 그려지는 곡선
-	AUC: ROC 곡선 결과를 하나의 숫자로서 나타낼 때는, ROC 곡선보다도 아랫 부분의 넓이를 이용
회귀:
-	평균제곱오차
-	제곱근평균제곱오차
-	평균절대오차

로지스틱 회귀
-	로지스틱 회귀는 선형모형이므로, 단순한 분류만 가능한 선형분류기

결정 트리와 랜덤 포레스트
-	설명변수에 대해 x>10이라면 가지A, 그렇지 않 다면 가지B와 같은 조건 분기로 트리 구조를 만들어 데이터를 분류하는 방법
-	이를 다수결로 수행 하는 랜덤 포레스트

신경망
-	여러 로지스틱 회귀의 출력이 또 다른 로지 스틱 회귀의 입력이 되는 모형
-	신경망을 이용한 학습은 신경 세포 사이의 가중치를 조정하는 것
-	오차역전파법 사용

수리 모형
- 현상의 프로세스나 메커니즘을 가정하고 그 움직임을 조사하는 모형
- 메커니즘을 수학적으로 나타내는 것에 서 출발하여 논리적으로 무엇이 일어나는가를 조사하는 방법
- 연역적인 방법
- 많은 경우 메커니즘에 기반을 둔 모형을 구축하기 때문에, 데이터로 경험하지 못한 범위라도 예측이나 통제가 가능할 때 존재

완벽한 모형은 없다
-	'현상의 실제 모습=실제 모형'을 만들기란 불가능
-	모형은 어디까지나 현상을 이해하는 데 도움이 되는 근사 도구로, 실제 현 상 그 자체와 모형은 다르다는 사실을 명심하는 것이 중요

수리모형
-	가상의 세계를 상정하고 어떤 규칙을 적용 했을 때 무엇이 일어나는가를 조사하는 방법
1)	결정론 모형: 미분 방정식, 차분 방정식, 편미분 방정식
2)	확률 모형: 무작위 행보, 마르코프 연쇄
-	결정론: 특정 시각의 상태가 정해지면 다음 시각의 상태는 단 하나로 정해지는 성질
-	모형의 복잡함과 이해의 용이함 간에는 상충 관계 존재

미분 방정식(차분 방정식)
-	파라미터를 다양하게 바꾸었을 때 어떤 일이 일어나는가를 조사할 수 있다는 것이 수리 모형의 강점

-	밀도효과를 적용한 모형: x가 늘어날수록 그 증가율이 줄어드는 효과
-	미분 방정식의 평형점을 조사하고, 그 평형점이 안정인지 불 안정인지를 확인함으로써, 그 모형의 성질 일부를 밝힐 수 있다
